요소 정렬 스타일(float/clear)

float : 요소를 띄워서 좌,우로 정렬하는 속성
clear : float로 인해 띄워져 있는 상태를 해제하는 속성.
왼쪽 정렬 해제 시 -> clear : left;
오른쪽 정렬 해제 시 -> clear : right;
양쪽 모두 해제 시 -> clear : both;
