선택자 우선순위

        기본적으로 CSS 속성은 style태그에 작성된 순서(위->아래)대로 해석되며
        같은 요소에 여러 CSS 속성이 설정된 경우 우선순위에 따라 스타일이 적용된다.

        1순위 : !important
        2순위 : inline-style 속성
        3순위 : id 선택자
        4순위 : class 선택자
        5순위 : tag 선택자

        같은 우선순위로 설정 CSS가 있을 경우
        마지막에 작성된 CSS 설정이 적용됨.
우선순위 테스트
:only-child -> 자식요소가 하나밖에 없을 때 적용 :only-of-type -> 같은 형제 요소가 없을 때 적용 :empty -> 자식이 없는 요소
