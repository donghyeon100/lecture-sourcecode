CSS란?

       Cascading Style Sheets의 약자로
       마크업 언어가 실제로 화면에 표시되는 방법(모양, 스타일)을
       기술하는 언어.
       (html 코드가 화면에 어떤 모양으로 보여질지 지정하는 언어)

       W3C 표준으로 지정되어있음.

CSS 기본 선택자

        - 선택자란?
        특정한 HTML 태그를 선택하는 기능으로,
        선택된 태그에 원하는 스타일 또는 기능을 적용할 수 있다.

태그 선택자(태그명)

        현재 HTML 문서 내에 같은 태그를 선택하는 선택자

        span { background-color : yellow; }

        위와 같은 형식으로 작성함.
태그선택자 span 태그 내부에 작성된 글씨 b태그 내부에 작성된 글씨 span 태그 내부에 작성된 글씨

아이디 선택자(#아이디속성값)

        HTML 문서 내에 해당하는 id 속성값을 가진 태그를 선택하는데 사용됨.
        (id 속성값은 페이지 내에서 유일해야 한다!!)    
        
        #id1{ color : red; }
        -> id 속성값이 "id1"인 태그의 글자색을 red로 설정

클래스 선택자(.클래스속성값)

        HTML 문서 내에 같은 클래스 속성값을 가지는 태그를 모두 선택할 때 사용.
        
        class 속성 : class 속성 값이 같은 태그들은 하나의 그룹으로 이루어짐.

        .클래스속성값{ color : blue; }
