기본 속성 선택자

        태그 내에 작성되는 내용 중
        태그명을 제외한 부분을 속성이라고 함.

        ex) <p id="test" class="cls1" name="p1" >
        
        id, class, name == 속성

        "test", "cls1", "p1" == 속성 값


        선택자[속성="속성값"]{ css 설정 코드; }

반응 선택자

        사용자의 움직임에 따라 달라지는 선택자

        - 사용자가 요소를 클릭하고 있는 경우  (:active)
        - 사용자가 요소에 마우스를 올려두는 경우 (:hover)

상태 선택자

        입력 양식의 상태에 따라 선택되는 선택자
        (checked, focus, enabled/disabled)
체크 상태에 따른 선택자 :checked

사용 불가능한 input 태그 선택자 :disabled

동위 선택자

        동위 관계(동일한 위치)에서 뒤에 위치한 요소를 선택하는 선택자
        
        바로 뒤에 하나 선택 :  선택자a + 선택자b{ 설정내용; }

        뒤에 있는 모든 요소 선택 : 선택자a ~ 선택자b{ 설정내용; }
