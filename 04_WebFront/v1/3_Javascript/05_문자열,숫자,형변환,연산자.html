문자열

        "" 또는 ''를 리터럴 가지고 있는 자료형.

        String이라는 자바스크립트 내장 객체가 존재하며
        문자열과 관련된 기본적인 함수(메소드)를 제공함.

숫자

        숫자형 리터럴 표기법 : 10(정수), 3.14(실수), Infinity(무한), NaN(Not a Number) 
        
        Math 라는 수학 관련 내장 객체가 존재함.

숫자 관련 형변환

동등, 동일 연산자

        동등 비교 연산자 (==, !=)
            ->   자료형 관계 없이 값이 같으면 true
            1 == "1" == true
            0 == "0" == false 

        동일 비교 연산자 (===, !==)
            -> 자료형과 값이 모두 같으면 true
