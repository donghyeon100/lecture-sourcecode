배열

배열이란?

자바스크립트에서는 별도의 자료형 지정이 없기 때문에
모든 자료형을 보관하는 변수의 모음을 배열로 처리함.
자바에서의 컬렉션과 비슷하다.

배열의 선언

배열 선언시 배열의 크기를 정하거나, 정하지 않고 선언할 수 있다.

배열 객체의 메소드(함수)

배열도 하나의 객체이기 때문에 배열에서 활용할 수 있는 메소드가 있음.

indexOf() : 배열에서 요소가 위치한 인덱스를 리턴

concat(배열명) : 다수의 배열을 결합

toString(), join() : 배열을 결합하여 문자열로 반환

toString()은 배열을 문자열로 합칠 때 요소 사이에 ,(콤마)를 추가하여 합침
join()도 기본적으로 toString()과 같지만 매개변수로 구분자를 지정할 수 있음.


sort() : 배열을 오름차순으로 정렬
reverse() : 배열의 순서를 뒤집음

sort(), reverse() 수행 시 원본 데이터의 순서 자체가 변함.


push() : 배열의 맨 뒤에 요소 추가
pop() : 배열의 맨 뒤에 요소 제거


shift() : 배열의 맨 앞에 요소 제거
unshift() : 배열의 맨 앞에 요소 추가


slice() : 배열의 요소 선택 잘라내기
splice() : 배열의 index위치의 요소 제거 및 추가
