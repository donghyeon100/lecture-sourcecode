해당 브라우저는 자바스크립트를 미지원 하거나 자바스크립트 실행 기능이 비활성화 되어 있습니다.

자바스크립트 개요

스크립트 언어란?

        기존에 이미 존재하는 프로그램의 동작을 사용자의 요구에 맞게 수행되도록 제어하는 용도의 언어.
        소스코드의 수정이 많은 곳에서 사용하기 적합. (인터프리터(Interperter) 방식)

        - 인터프리터란? 
            소스코드를 한줄 씩 읽어 바로 실행하는 방식.

        매우 빠르게 배우고 짧은 소스코드 파일로 상호작용 하도록 고안되었다.

Javascript

        웹 브라우저에서 많이 사용하는 인터프리터 방식의 객체지향 프로그래밍 언어.
        HTML 문서내에 기술되고, HTML과 함께 웹 브라우저에 의해 해석됨

        ECMA 스크립트 표준을 따르는 대표적인 웹 기술이다.

자바스크립트 작성 방법

        소스는 함수 단위로 작성하고, 이벤트속성을 사용해서 구동되게 함.
        
        이벤트 속성은 태그마다 차이가 있음
        on이벤트명="실행할 함수명([전달값])"

inline 방식

태그에 직접 간단소스코드를 작성해서 실행되게 하는 방법

마우스오버 / 마우스아웃

internal 방식

script 태그 영역에 작성해서 실행되게 하는 방법
head태그, body태그에 작성 가능

external 방식

별도의 js파일로 작성해서 가져다 사용하는 방법
