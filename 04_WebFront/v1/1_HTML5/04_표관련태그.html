table 태그

웹 문서에서 자료 정리를 위해 주로 사용되는 태그
행과 열로 이루어져 있으며, 행과 열이 교차되는 지점을 셀(cell)이라 함.

tr 태그

Table Row의 약자로 한 행을 나타내는 태그.

td 태그

Table Data의 약자로 한 컬럼값(== 셀)을 나타내는 태그.

th 태그

Table Header의 약자로 컬럼에 대한 제목을 표시하는 태그
가운데 정렬 + 굵은 글씨(Bold)로 글이 작성됨

caption 태그

테이블의 제목이나 내용을 추가하는 태그

웹 브라우저의 종류
브라우저명 제조사 홈페이지 주소
익스플로러 MS https://www.microsoft.com

사파리 Apple https://www.apple.com
파이어폭스 Mozilla https://www.mozilla.org

rowspan, colspan

td 또는 th 태그에 작성하는 속성.
rowspan : 행 병합
colspan : 열 병합
