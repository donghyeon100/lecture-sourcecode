문단을 구분하는 태그(p, pre)

p, pre 공통점 : 글을 작성하는 용도의 태그
차이점 : p태그는 문단을 구분하는 용도
pre태그는 문단 구분 + 작성된 모양 그대로를 표현하는 태그

글자 관련 기타 태그

b태그 : 단순히 글자를 굵게 표시하는 태그
strong 태그 : 글자를 강조하는 태그(웹 접근성)
i태그 : 단순히 글자를 기울이는 태그
em 태그 : 글자를 강조하는 태그(웹 접근성)
mark 태그 : 형광펜 효과를 나타내는 태그
u 태그 : 밑줄을 긋는 태그
small태그 : 글자를 작게 나타내는 태그
sup, sub 태그(윗첨자, 아래첨자) : x2 , log2
s태그 : 취소선을 긋는 태그
abbr 태그 : 마우스 오버 시 툴팁 형태로 출력되는 태그
HTTP

글자 관련 태그 응용

태그들은 해당 태그 내에서 중첩으로 사용 가능하다.
굵은글자를 사용할 수 도 있고, 기울이거나, 취소선을 넣는 등 다양하게 사용할 수 있다.
