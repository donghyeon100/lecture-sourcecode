input 태그 : 웹 문서에서 사용자가 입력할 수 있는 양식을 제공하는 태그.

        form 태그 :  내부에 작성된 input 태그 값을 서버로 넘기는 역할을 하는 태그.

        * form 태그 속성
        1) action : form 태그 내부에 입력된 값을 전달할 페이지 url을 작성하는 속성.
        2) method : 데이터 전송 방식 지정
        3) name : form 태그의 이름 지정 속성. form태그 구분을 위해 사용
        4) target : action에 지정된 페이지를 새탭/현재창 중 어디서 열지 선택
