<img> 태그

        img 태그 : 웹 문서에 사진이나 그림을 삽입할 때 사용하는 태그

        img 태그 속성
        1) src : 삽입할 이미지 경로를 지정하는 속성
        2) width : 이미지의 너비(폭, 가로 크기)를 지정하는 속성
        3) height : 이미지의 높이(세로 크기)를 지정하는 속성
        4) alt : 이미지를 설명하는 속성. 이미지가 출력이 되지 않는 경우 화면에 표시됨.

src 속성 확인


alt 속성 확인

width, height 속성 확인

고정 크기 단위 : 화면 크기 관계 없이 지정된 값으로 크기가 지정되는 단위
가변 크기 단위 : 화면이나 영역의 기준 사이즈에 따라 크기가 변경되는 단위
