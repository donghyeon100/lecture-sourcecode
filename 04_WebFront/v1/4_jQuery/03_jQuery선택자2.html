prop() 메소드

        attribute : type, name, class, id, value 등과 같이
                    속성값을 별도로 입력해야되는 속성

        property : checked, selected 와 같이
                   속성값이 별도로 입력되지 않는 속성


        prop("속성명")  :  해당 속성이 요소에 존재하면 true, 아니면 false

        prop("속성명", ture | false) : 해당 속성을 checked 또는 selected
                                        상태로 변경(true)/해제(false)
취미 : 축구 게임 음악감상
